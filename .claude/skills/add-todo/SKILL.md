# TODO追加フロー

ユーザーからTODOに追加したい内容を受け取り、CLAUDE.mdに記載してください。

## 手順

1. ユーザーの入力内容を確認する
   - `$ARGUMENTS` があればそれをTODO内容として使う
   - なければ「TODOに追加したい内容を教えてください」と聞く

2. CLAUDE.md の `### Phase 2` セクション内の既存TODO一覧を Read で確認する

3. ユーザーの入力を以下のルールで整理する:
   - 各項目を `- [ ] タイトル` 形式にまとめる
   - 補足説明があれば箇条書きのサブ項目として追加
   - 既存のTODOと重複する場合は、既存項目に追記する形でマージする
   - 関連する項目はグループ化する

4. CLAUDE.md の Phase 2 セクション内、適切な位置に Edit で追記する
   - 既存の未実装項目（`- [ ]`）の並びに挿入
   - 「予測入力機能」「差分比較機能」の前に配置（これらは大型機能のため末尾に置く）

5. 追加した内容をユーザーに報告する
