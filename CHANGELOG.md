# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [Unreleased]

## [0.1.4] - 2026-03-22

### Fixed
- コンポーネントのLR方向が保存・復元されないバグを修正 (#1, #2)
- 折りたたみコンポーネントの選択時にフォーカス枠が表示されない問題を修正 (#5)
- コンポーネントリサイズ時に内部ノードのサイズが連動して変わる問題を修正 (#17)
- コンポーネントの左右ゴーストノード接続時に線が表示されない問題を修正 (#18)
- ブリッジエッジの接続先ハンドルが方向に応じて正しく設定されない問題を修正
- 折りたたみコンポーネントのSVG縦線がリサイズで変動する問題を修正

### Changed
- コンポーネントパネルにTD/LRラベル表示を追加
- ノード/コンポーネントタブにアイコンを追加
- 折りたたみコンポーネントのリサイズ対応（サイズ保持・復元）
- コンポーネント展開時にzIndexを自動引き上げ（子ノードが他ノードに隠れない）
- ゴーストノード表示をコンポーネントのフロー方向に制限（TD: 上下のみ、LR: 左右のみ）

## [0.1.3] - 2026-03-20

### Added
- エクスポートダイアログ新設（未配置コンポーネント定義を含めるトグル付き） (#21)

### Fixed
- 右クリックドラッグによるキャンバスパンが動作しない不具合を修正 (#24)
- 全ノード削除後もMermaidコードにエッジが残る不具合を修正 (#21)
- ゴーストノードへのエッジ接続がストアに保存される不具合を修正 (#7, #16)
- 最背面設定ノードが選択時に最前面に浮き上がる不具合を修正 (#8)
- 内部記憶ノードのリサイズ時に十字線の幅が変動する不具合を修正 (#19)
- 定義済み処理ノードのリサイズ時に縦線の幅が変動する不具合を修正 (#19)

### Changed
- OUTPUTパネルでキャンバスに未配置のコンポーネント定義を非表示に変更 (#21)

## [0.1.2] - 2026-03-19

### Changed
- ノード右クリックメニューの「複製」を「形状変更」サブメニューに置換（アイコン付き15種） (#6)

### Fixed
- コンテキストメニューから文字色をデフォルトに戻せないバグを修正 (#11)
- コンテキストメニューからエッジ線色をデフォルトに戻せないバグを修正 (#11)
- エッジ選択時・エッジ上での右クリックでコンテキストメニューが表示されないバグを修正 (#12)

## [0.1.1] - 2026-03-17

### Added
- β版ポップアップとGitHub Issuesボタン (#4)

### Fixed
- エッジの矢印（マーカー）が表示されない不具合を修正 (#10)
- エッジの線色変更時に矢印の先端のみ変わり線本体に反映されない不具合を修正 (#10)
- ゴーストエッジに矢印マーカーが付与されていない不具合を修正 (#10)
- BulkEdit Categoryモードのノード/エッジ表示順をキャンバス位置順に変更 (#3)

## [0.1.0] - 2026-03-16

Initial release.

### Added
- キャンバス上にノードを自由配置（ドラッグ&ドロップ）
- ノード形状 15種類
- リアルタイムMermaid記法生成
- 独自フォーマット（.flowmaid）のエクスポート/インポート
- Mermaid記法からの逆変換
- undo/redo（50履歴）
- コピー&ペースト
- コンポーネント（再利用可能テンプレート）
- テキスト一括編集（BulkEdit）モード
- 予測入力（ゴーストノード）
- ダーク/ライトテーマ
- 多言語対応（日本語/英語）
- ミニマップ
- 操作ガイドパネル
