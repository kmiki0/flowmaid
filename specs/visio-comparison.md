# Visio vs Flowmaid 機能比較

Visio の主要機能と Flowmaid の現状を比較し、不足分をカテゴリ別にまとめたもの。

凡例: ✅ 実装済み / 🔶 部分的に実装 / ❌ 未実装

---

## 図形 (Shapes)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 数百種類のシェイプ | ✅ 15種類（矩形、菱形、角丸、円、平行四辺形、円柱、六角形、スタジアム、台形、書類、定義済み処理、手入力、内部記憶、表示、テキスト） |
| シェイプのグループ化 | ❌ なし |
| サブプロセス（ノード内に別フローを格納） | ✅ コンポーネント機能（テンプレート定義＋インスタンス配置） |
| シェイプの回転 | ❌ なし |
| シェイプの反転（左右・上下） | ❌ なし |
| シェイプ内に画像やアイコン挿入 | ❌ なし |
| カスタムシェイプの作成 | ❌ なし |
| シェイプのロック（移動・編集禁止） | ❌ なし |
| シェイプのリサイズ | ✅ NodeResizer で対応 |

## 接続線 (Connectors)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 直線・折れ線・曲線の選択 | ✅ ベジェ / 直線 / 折れ線（右クリックメニューで切替） |
| 矢印の始点・終点スタイル（矢印、丸、なし等） | ✅ 矢印（閉じ）/ 矢印（開き）/ なし（始点・終点それぞれ設定可） |
| 接続線の経路の手動調整（ウェイポイント） | ✅ 折れ線エッジのwaypoint編集対応 |
| コネクタの自動ルーティング（障害物回避） | 🔶 折れ線エッジのみ自動回避（ベジェ・直線は未対応） |
| 双方向矢印 | ✅ 始点・終点の両方に矢印を設定可能 |
| 線の太さ | ✅ 細い(1px) / 標準(2px) / 太い(4px) |
| 線の色 | ✅ 10色選択可 |
| 線種（点線、破線等） | ✅ 実線 / 破線 / 点線 |

## スタイル・書式 (Formatting)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 塗りつぶし色 | ✅ 10色選択可（フォーマットバー、透明度・明暗調整対応） |
| グラデーション | ❌ なし |
| 枠線の色 | ✅ 10色選択可（フォーマットバー、透明度・明暗調整対応） |
| 枠線の太さ | ✅ 細い(1px) / 標準(2px) / 太い(4px) |
| 枠線のスタイル | ✅ 実線 / 破線 / 点線 |
| フォントの種類・サイズ・色・太字・斜体 | 🔶 サイズ4段階・色10色・太字・斜体・下線・文字位置対応（フォント種類は未対応） |
| テーマ・配色セットの適用 | 🔶 ダーク/ライト切替のみ |
| 条件付き書式（データに応じた色分け） | ❌ なし |
| 透明度の設定 | ✅ 塗りつぶし・枠線・文字・エッジそれぞれ0〜10段階 |

## レイアウト・配置 (Layout & Alignment)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 自動レイアウト（ツリー、フローチャート等） | ❌ なし |
| スナップ（グリッド、他の図形へのスナップ） | ✅ スナップ対応 |
| ガイドライン（整列補助線） | ✅ ガイドライン表示対応 |
| 図形の整列（左揃え、中央揃え等） | ✅ 左/中央/右/上/中央/下 の6方向 |
| 等間隔分布 | ✅ 水平 / 垂直（3ノード以上） |
| グリッド表示・サイズ設定 | 🔶 表示のみ（設定なし） |
| レイヤー管理（表示/非表示、ロック） | ❌ なし |
| Z順序（前面/背面への移動） | ✅ 最前面 / 一つ前 / 一つ後ろ / 最背面 |

## テキスト・注釈 (Text & Annotations)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| ノード内の複数行テキスト | ✅ Shift+Enter で改行対応 |
| フリーテキスト（キャンバス上に直接配置） | ✅ テキストノード（枠線・背景なし、Mermaid出力から除外） |
| コメント・注釈の追加 | ❌ なし |
| テキストの回転 | ❌ なし |
| リッチテキスト（リンク、リスト等） | ❌ なし |

## データ連携 (Data)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 図形にカスタムプロパティ（メタデータ）付与 | ❌ なし |
| Excel / DB からのデータインポート・リンク | ❌ なし |
| データに基づく自動図形生成 | ❌ なし |
| レポート生成 | ❌ なし |

## ページ・ドキュメント (Pages & Documents)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 複数ページ（タブ切替） | ❌ 1ページのみ |
| ページサイズ・向き設定 | ❌ なし |
| 背景ページ（共通背景・ヘッダー/フッター） | ❌ なし |
| ハイパーリンク（シェイプ間、外部URL） | ❌ なし |

## エクスポート・共有 (Export & Sharing)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| PNG / SVG / PDF エクスポート | ❌ なし |
| Visio / VDX 形式の保存 | 🔶 .flowmaid 独自形式のみ |
| 印刷（ページ設定、拡大縮小） | ❌ なし |
| リアルタイム共同編集 | ❌ なし |
| 共有リンク | ❌ なし |

## 操作・UX (Interaction)

| Visio の機能 | Flowmaid の現状 |
|---|---|
| 右クリックコンテキストメニュー | ✅ ノード・エッジ・キャンバスで対応 |
| ドラッグで範囲選択 | ✅ React Flow 標準 |
| コピー & ペースト（Ctrl+C / Ctrl+V） | ✅ ノード＋関連エッジをコピー |
| 検索・置換 | ❌ なし |
| ステンシル（テンプレートパレット）の管理・カスタマイズ | 🔶 固定パレット（15種類）+ コンポーネントタブ |
| マクロ / スクリプト | ❌ なし |
| 最近使ったファイル一覧 | ❌ なし |
| 多言語対応 | ✅ 日本語 / 英語（ブラウザ言語自動検出 + 手動切替） |
| undo / redo | ✅ Ctrl+Z / Ctrl+Shift+Z（50履歴） |
| キーボードショートカット | ✅ undo/redo, コピー/ペースト, エクスポート, Delete, Ctrl+A全選択 |

## 実装済み機能サマリー（Phase 1〜2 完了分）

1. ✅ **ノード形状 15種類**（処理、判断、代替処理、結合子、データ、データベース、準備、端子、手操作、書類、定義済み処理、手入力、内部記憶、表示、テキスト）
2. ✅ **接続線**（ベジェ/直線/折れ線、矢印スタイル、線種、太さ・色、waypoint編集）
3. ✅ **スタイル設定**（塗りつぶし・枠線・文字 各10色、透明度・明暗調整）
4. ✅ **フォント設定**（サイズ4段階、色10色、太字・斜体・下線、文字位置）
5. ✅ **レイアウト支援**（整列6方向、等間隔分布、スナップ、ガイドライン、Z順序）
6. ✅ **テキスト**（複数行対応、フリーテキスト配置）
7. ✅ **コンポーネント**（テンプレート定義+インスタンス、展開/折りたたみ、ブリッジエッジ）
8. ✅ **操作**（コピペ、undo/redo、右クリックメニュー、キーボードショートカット）
9. ✅ **エクスポート/インポート**（.flowmaid形式、Mermaid逆変換）
10. ✅ **折れ線エッジのノード回避ルーティング**
11. ✅ **多言語対応**（日本語/英語）
12. ✅ **ダーク/ライトテーマ**

## 未実装で優先度が高そうなもの

1. **PNG / SVG / PDF エクスポート**
2. **自動レイアウト**
3. **検索・置換**
4. **シェイプのグループ化**
